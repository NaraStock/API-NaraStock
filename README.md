# NaraStock API Capstone Project DBS Foundation
## Prequiesete
- NodeJS

## Getting Started:
### Step 1 : Clone Repo
```
https://github.com/NaraStock/API-NaraStock.git
```

### Step 2 : Download all dependencies
```
npm install
```

### Step 3 : Run the server
```
# for production
npm run start

# for development
npm run start:dev
```
